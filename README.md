# 아파트 매물 서비스

## 프로젝트 기간
2022.11.16 ~ 2022.11.24


## 개요
부동산 매물 서비스로서, 주택 실거래가 내역을 제공하는 서비스입니다.<br>


## 개발 환경
DB: mysql 8.0.3.0<br>
BackEnd: Spring boot, JAVA 8<br>
Frontend: Vue.js<br>
협업: Git, Notion, Mattermost<br>
